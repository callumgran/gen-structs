# gen-structs
Generic abstract datastructures in C with extended functionality

### Datastructures
- [x] list (list_t)
- [x] queue (queue_t)
- [x] stack (stack_t)
- [x] binary search tree (tree_t)
- [x] singly linked list (s_linkedlist_t)
- [x] double linked list (d_linkedlist_t)
- [ ] binary heap
- [ ] fixed size hashtable
- [ ] dynamic hashmap
